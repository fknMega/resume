Nesher.io יצא ל-waitlist

Bun + Elasticsearch + MongoDB, פתיחה מדורגת והקשחה נגד שימוש זדוני.

Nesher.io Waitlist Search

Stack

Stack: Bun · Elasticsearch · MongoDB

שחררתי את Nesher.io ל-waitlist כדי לשמור על קצב איכותי ולהקשיח את המערכת מול שימוש זדוני.

  • Elasticsearch כ-DB לחיפוש: אינדוקסים יעודיים לביצועים נמוכים מ-100ms.
  • MongoDB למטא-דאטה של דליפות ומשתמשים: הפרדה ברורה בין חיפוש לנתוני הרשאות.
  • Bun לשרת API מהיר וקליל עם footprint מינימלי.
  • חומות אש, rate limiting וחסימת מדינות עוינות כבר בשכבת הקצה.
# חימום אינדקסים אחרי דיפלוי
curl -XPOST https://api.nesher.io/admin/warm-indexes \\
  -H "Authorization: Bearer $TOKEN"
ל-waitlist הצטרפו 894 קרדיטים ראשונים; פותחים גישה מדורגת עם ניטור צמוד.

המשך ייעול ותרשימי זרימה מלאים יפורסמו עם פתיחת הדלת הראשית.
